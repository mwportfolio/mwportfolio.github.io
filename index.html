ICT Supplier Analysis

This app uses Pandas, urllib and JSONLines Python libraries within Jupyter Notebooks to extract data from a public site and store it in a NoSQL database hosted on Google Cloud Platform (GCP) DataStore.

The Flask web application hosted on GCP AppEngine uses Jinja2 templates to present the data with Bootstrap and D3.
